# 📊 PabbleAI Option Chain Analyzer

NSE Option Chain ka real-time analysis tool

## 🚀 Features
- Live PCR (Put Call Ratio) calculation
- Max Pain strike finder
- Support & Resistance levels (Top OI strikes)
- Bullish/Bearish/Neutral signal
- NIFTY, BANKNIFTY, FINNIFTY support

## 🛠️ Tech Stack
- Python
- NSE API
- Pandas

## 📦 Installation
```bash
pip install -r requirements.txt
```

## ▶️ Usage
```bash
python option_chain.py
```
